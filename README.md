# MEM Telegram Bot

Игровой телеграм бот с внутренней валютой MEM.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` и добавьте:
```
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_ID=your_admin_id_here
CHANNEL_USERNAME=@InitialNFT
```

3. Запустите бота:
```bash
python main.py
```

## Команды бота

### Основные команды (без слеша):
- `Б/баланс` или `Баланс` - показать баланс
- `Банк` - показать баланс банка
- `Профиль` - информация о профиле
- `Инвентарь` - ваши предметы
- `Топ` - топ 10 игроков
- `Топ банки` - топ 10 банков
- `Лидерборд` - топ 5 за день
- `Бонус` - получить бонус (требуется подписка на @InitialNFT)
- `Помощь` - меню помощи

### Переводы:
- `Дать [сумма]` - перевести MEM (в ответ на сообщение)

### Операции с банком:
- `Банк снять [сумма]` - снять MEM с банка
- `Банк пополнить [сумма]` - положить MEM в банк

### Управление профилем:
- `Сменить аватарку` - сменить фото профиля
- `Б закрыть` - скрыть баланс в профиле
- `Б открыть` - показать баланс в профиле

### Админ команды:
- `Banuser` - забанить пользователя
- `Unbanuser [ID]` - разбанить пользователя
- `Игроконтроль` - управление играми
- `Выдать [сумма]` - выдать MEM пользователю
- `Забрать [сумма]` - забрать MEM у пользователя

## Особенности

- Все команды работают без слеша на русском языке
- Внутренняя валюта MEM
- Система банковских счетов
- Профили с аватарками
- Топ рейтинги
- Админ функции

## База данных

Используется SQLite с автоматическим созданием таблиц при первом запуске.
